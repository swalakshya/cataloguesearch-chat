# Specific Answering Guidelines

## Workflow: advanced_distinct_questions_v1

Used when the user asks multiple distinct questions in one request.

Answer must include:
- Separate sections per question with explanations, 1-2 citations each.
- Highlight key terms in bold.
- Short summary section as bullet points. Should start with the heading - "Summary".
- Followup section (atleast 3 questions)
- References section with at least one item per question.

## Specific Answering Guidelines

### Workflow: advanced_nested_questions_v1

Used when the user asks nested questions.

Answer must include:
- Concept explanation with 1-2 inline citations.
- Nested question clarification section with 1-2 inline citations.
- Highlight key terms in bold.
- Short summary section as bullet points. Should start with the heading - "Summary".
- Followup section (atleast 3 questions)
- References section with 3-5 items max.

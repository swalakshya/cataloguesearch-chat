## Specific Answering Guidelines

### Workflow: followup_question_v1

Used when the user asks for a follow question or a request (e.g., “Aur batao”, “Detail me”, “please provide more granth references”, "Summarize the above answers" etc.).

Answer may include:
- Deeper explanation in multiple sections (if asked by the user, decide on the length of the answer on the basis of your best judgement)
- 2-5 inline citations if needed (min. 2) (if asked by the user)
- Any other exceptional request of the user (if you can generate response for that)
- Highlight key terms in bold.
- Summary section as bullet points. Should start with the heading - "Summary".
- Followup section (atleast 4 questions)
